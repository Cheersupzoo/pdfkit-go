# pdfkit-go

Pure Go library for creating and manipulating PDFs, inspired by [PDFKit](https://github.com/foliojs/pdfkit) and [pdf-lib](https://github.com/Hopding/pdf-lib).

**Policy:** pure Go only (dependencies allowed, no CGO). Prefer stdlib; use mature pure-Go packages for fonts/WOFF2 and similar.

**Status:** research and planning. See [docs/RESEARCH_AND_PLAN.md](docs/RESEARCH_AND_PLAN.md).
