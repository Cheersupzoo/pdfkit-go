# pdfkit-go

Pure Go library for creating and manipulating PDFs, inspired by [PDFKit](https://github.com/foliojs/pdfkit) and [pdf-lib](https://github.com/Hopding/pdf-lib), aiming to stay as close to the Go standard library as practical.

**Status:** research and planning. See [docs/RESEARCH_AND_PLAN.md](docs/RESEARCH_AND_PLAN.md).
